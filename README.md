# useless-web-website

## About the Project
This **useless-web-website** is inspired by the useless web, which serves the purpose of curing the boredem of it's user. By pressing enter, the following happens:
1. the page shows a random generated background color
2. the click counter of the enter key increases once and displays the new click counter in the frontend
3. the current background color displayed in the frontend is updated to the new hex color code
